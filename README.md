# Raj Anand Jha — Portfolio

A responsive static portfolio built with HTML, CSS, and JavaScript. No package installation or build step is required.

## Run locally

From this folder, run:

```bash
python -m http.server 8000
```

Then open `http://localhost:8000`.

You can also open `index.html` directly, though a local server is recommended for consistent browser behavior.

## Customize

- Replace the two editable project cards in `index.html` with real case studies and links.
- Add verified education, work experience, and certifications in the Journey section.
- Add a real GitHub or LinkedIn URL to the footer when available.
- The contact form uses the existing Web3Forms key from the original site. Replace it if the form owner changes.
